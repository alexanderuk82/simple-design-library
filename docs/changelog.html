v0.1.0 December 28, 2024

Initial Beta Release

The first public beta of Simple Design System. This release includes the foundation of the design system with core tokens, CSS variables, and the documentation website.

  • New Design tokens system with color primitives and semantic colors
  • New Light and Dark mode support with OKLCH color space
  • New Typography scale with responsive sizing
  • New Spacing and size tokens
  • New Landing page with GSAP animations
  • New GitHub Pages deployment with auto-deploy
  • New Changelog page
