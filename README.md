# Simple Design Library

A lightweight, customizable UI component library for modern web applications.

> **Note:** This project is under active development. Features and APIs may change.

## Overview

Simple Design Library provides a collection of beautifully crafted, accessible UI components that work seamlessly across different frameworks and vanilla HTML/CSS projects.

## Features (Planned)

- **Framework Agnostic** - Works with React, Vue, Svelte, or vanilla HTML/CSS
- **Lightweight** - Minimal footprint, maximum performance
- **Customizable** - Easy theming with CSS variables
- **Accessible** - WCAG 2.1 compliant components
- **TypeScript Ready** - Full TypeScript support out of the box
- **Dark Mode** - Built-in dark mode support
